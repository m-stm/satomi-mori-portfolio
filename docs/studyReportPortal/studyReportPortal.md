# 学習レポート共有サイト

## プロジェクト概要

福祉事業所内で事業所全体のスキル向上と、利用者同士のコミュニケーションのきっかけ作りを行う目的で、利用者自らの学習レポート（成果物や共有事項）を投稿・閲覧管理が可能な会員制webサイトの開発プロジェクト

## プロジェクト誕生の背景

PDFファイルの投稿管理アプリケーションの経験を活かして、何か問題解決ややり良くできることがないか考えている時に、「学習成果をアウトプットしたり、他の人の学習成果からヒントを得て自分自身の学習に活用していくことができたらいいな」という想いになりました。<br/>
そこから、現状の手段と福祉事業所という特有の環境上必要な配慮事項は何かを考え、本プロジェクトを企画しました。

| :--------------: | :-------------------------------------------- |

| Backend（補助）  | Google Apps Script（GAS）、サービスアカウント |

|     ドメイン     | 機能                                 |

| 学習レポート管理 | 一覧取得、登録、編集、削除           |

**「UI」にこだわり、「ユーザー体験(UX)」を高める事意識してwebサイトを構成・デザインしてきました。**<br/>

|:---:|:---|:---|:---|
|**1**|迷わないフォームの注釈情報|登録・編集フォームは、注釈やプレースホルダを活用して、入力のルールや例示を表示。<br/>どんなルールで何を入力したらいいのかわかりやすく丁寧に表現|<img src="./images/study_form.png" width="100%">|
