# Satomi Mori - Portfolio Repository

本リポジトリは、私の開発実績、および「現場の課題をロジカルに解決する」ために作成した各システムの【業務要件定義書・設計ドキュメント】を一般公開・資産管理している場所です。

- 🌐 **ポートフォリオサイト本体**: [URL](https://m-stm.github.io/satomi-mori-portfolio/)
